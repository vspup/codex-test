# Test Runners

Этот каталог содержит самостоятельные сценарии для проверки и прожига оборудования.

## Скрипты

### `burn_in_runner.py`
Минимальный клиент Electabuzz, который подключается к устройству, включает силовые модули, управляет режимом Main Mode и периодически опрашивает ключевые датапоинты (токи, напряжения, температуры). Дополнительно поддерживает подключение к мультиметру Hioki DM7275 и ведёт CSV-лог измерений.

**Запуск:**
```bash
python -m py.test_runners.burn_in_runner
```

Команду следует выполнять из корня репозитория, чтобы гарантировать корректную работу относительных импортов.
При необходимости укажите альтернативный путь к интерпретатору Python и убедитесь, что у скрипта есть доступ к последовательному порту мультиметра.
